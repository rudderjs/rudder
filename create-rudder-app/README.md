# create-rudder-app

> **The scaffolder is now [`create-rudder`](https://www.npmjs.com/package/create-rudder).** This package is a maintained alias kept around for old install commands cached in blog posts and muscle memory — it works identically and is published in lockstep, but new docs and marketing use the shorter name. Switch when you can:
>
> ```bash
> npm create rudder@latest my-app          # new
> npm create rudder-app@latest my-app      # still works (this package)
> ```

Source of truth: <https://github.com/rudderjs/rudder/tree/main/create-rudder>.

For documentation, install instructions, and contributing details see the [`create-rudder` README](https://github.com/rudderjs/rudder/blob/main/create-rudder/README.md).
