# Panels v2 Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Evolve `@boostkit/panels` into a production-grade admin panel system inspired by the best of Filament PHP and Payload CMS, with a polished shadcn/ui interface and two layout options (sidebar + topbar).

**Architecture:** The package stays split between a TypeScript backend (`packages/panels/src/`) that defines resources/fields/actions and mounts API routes, and React UI pages published to the app's `pages/(panels)/` directory via `vendor:publish`. New field types add both a backend Field class and a corresponding `FieldInput` rendering case. Layout options are controlled via `Panel.layout('sidebar' | 'topbar')`.

**Tech Stack:** TypeScript (NodeNext ESM), React 19, shadcn/ui (base-ui primitives), Tailwind CSS v4, `@boostkit/router`, `@boostkit/orm`, `@boostkit/storage`

if (search && searchableColumns.length > 0) {
  // Chain a where for each searchable column — ORM supports multiple where() calls
  // Use LIKE operator for partial matching
  q = q.where(searchableColumns[0]!, 'LIKE', `%${search}%`)

### Task 3: Wire SelectFilter into the list API endpoint

**Problem:** `filters()` method on Resource is defined but filter values from `?filter[status]=active` are ignored.

**Files:**
- Modify: `packages/panels/src/PanelServiceProvider.ts` — list route handler

**Step 1: Parse filter params and apply**

```ts
const filters = resource.filters()
for (const filter of filters) {
  const filterMeta = filter.toMeta()
  const value = url.searchParams.get(`filter[${filterMeta.name}]`)
  if (value !== null && value !== '') {
    const applied = filter.apply(value)
    // applied is { column: value } for SelectFilter
    for (const [col, val] of Object.entries(applied)) {
      if (col === '_search') {
        // SearchFilter returns { _search: { value, columns } }
        const { value: sv, columns } = val as { value: string; columns: string[] }
        if (columns[0]) q = q.where(columns[0], 'LIKE', `%${sv}%`)
        for (let i = 1; i < columns.length; i++) {
          q = q.orWhere(columns[i]!, `%${sv}%`)
        }
      } else {
        q = q.where(col, val)
      }
    }
  }
}
```

A `Section` is not a field — it's a layout wrapper. It has a title, optional description, optional collapsible flag, and contains an array of fields.

**Backend:**

```ts
// packages/panels/src/Section.ts
export interface SectionMeta {
  type:         'section'
  title:        string
  description?: string
  collapsible:  boolean
  collapsed:    boolean
  columns:      1 | 2 | 3
  fields:       FieldMeta[]
}

export class Section {
  private _title:        string
  private _description?: string
  private _collapsible:  boolean = false
  private _collapsed:    boolean = false
  private _columns:      1 | 2 | 3 = 1
  private _fields:       Field[] = []

  static make(title: string): Section {
    const s = new Section()
    s._title = title
    return s
  }

  description(text: string): this  { this._description = text; return this }
  collapsible(val = true): this    { this._collapsible = val; return this }
  collapsed(val = true): this      { this._collapsed = val; return this }
  columns(n: 1 | 2 | 3): this     { this._columns = n; return this }
  schema(...fields: Field[]): this { this._fields = fields; return this }

  // For service provider to extract flat fields for validation
  getFields(): Field[] { return this._fields }

  toMeta(): SectionMeta {
    return {
      type:        'section',
      title:        this._title,
      description:  this._description,
      collapsible:  this._collapsible,
      collapsed:    this._collapsed,
      columns:      this._columns,
      fields:       this._fields.map(f => f.toMeta()),
    }
  }
}
```

export class Tab {
  constructor(
    public readonly label: string,
    public readonly fields: Field[],
  ) {}

  toMeta() {
    return { label: this.label, fields: this.fields.map(f => f.toMeta()) }

export class Tabs {
  readonly type = 'tabs'
  private _tabs: Tab[] = []

  static make(): Tabs { return new Tabs() }

  tab(label: string, ...fields: Field[]): this {
    this._tabs.push(new Tab(label, fields))
    return this
  }

  getFields(): Field[] { return this._tabs.flatMap(t => t.fields) }

  toMeta() {
    return { type: 'tabs', tabs: this._tabs.map(t => t.toMeta()) }
  }
}

// In PanelServiceProvider.boot() — mount upload endpoint
router.post(`${apiBase}/_upload`, panelMw, async (req, res) => {
  const { Storage } = await import('@boostkit/storage')
  const formData    = await req.raw.formData()
  const file        = formData.get('file') as File
  const disk        = String(formData.get('disk') ?? 'local')
  const directory   = String(formData.get('directory') ?? 'uploads')

  const ext      = file.name.split('.').pop()
  const filename = `${Date.now()}-${Math.random().toString(36).slice(2)}.${ext}`
  const path     = `${directory}/${filename}`

  await Storage.disk(disk).put(path, Buffer.from(await file.arrayBuffer()))
  res.json({ url: await Storage.disk(disk).url(path), path })
})
```

        const file = e.target.files?.[0]
        if (!file) return
        const fd = new FormData()
        fd.append('file', file)
        fd.append('disk', field.extra.disk)
        fd.append('directory', field.extra.directory)
        const res = await fetch(`${apiBase}/_upload`, { method: 'POST', body: fd })
        const { url } = await res.json()
        onChange(url)
      }} />
    </div>
  )
}
```

## Phase 6 — Global Search & Dashboard

### Task 20: Global search endpoint

Mount `GET /api/_search?q=foo` on each panel that searches all searchable resources and returns grouped results.

```ts
router.get(`${apiBase}/_search`, panelMw, async (req, res) => {
  const q = new URL(req.url, 'http://x').searchParams.get('q') ?? ''
  if (!q) return res.json([])

  const results = await Promise.all(
    panel.getResources().map(async (ResourceClass) => {
      const resource = new ResourceClass()
      const searchableCols = resource.fields()
        .filter(f => f.isSearchable())
        .map(f => f.getName())
      if (!searchableCols.length || !ResourceClass.model) return null

      const records = await ResourceClass.model
        .query()
        .where(searchableCols[0]!, 'LIKE', `%${q}%`)
        .limit(5)
        .get()

      return { resource: ResourceClass.getSlug(), label: ResourceClass.getLabel(), records }
    })
  )

  res.json(results.filter(Boolean))
})
```
